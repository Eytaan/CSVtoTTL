# CSVtoTTL
#Document d'installation

1. Télécharger le dossier zippé sur GitHub à l'adresse suivante https://github.com/Eytaan/CSVtoTTL.git
2. Cliquer sur "Code" --> "Download Zip"
3. Dans votre réperoire, dézipper le dossier
4. Ouvrez un terminal, placez vous dans le bon dossier (dossier dézippé précédemment)
5. Taper la commande "python code.py"
